# PYRDLE 

## Overview

- Wordle in Python
- Only 5 word Wordle can be played ( ... I'm willing to support other numbers of letters.)

## Requirement

- wsl2
- python = "^3.8"
- numpy = "^1.22.3"
- poetry-core = "^1.0.0"


## Usage

- please command "python sample.py"
- if you want to know the correct answer before playing "python sample.py -d"
